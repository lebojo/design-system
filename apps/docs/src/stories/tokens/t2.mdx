# T2 Semantic Tokens

## What are T2 Tokens?

T2 tokens are semantic design tokens that reference T1 (primitive) values. They define aliases and common names to abstract the T1 tokens.

### Characteristics:

- **Purpose-driven naming**: Named by what they do, not what they are
- **Resolve to T1 values**: Point to primitive tokens for actual values
- **Modifiers**: Modifiers could override T2 tokens to adapt to a specific use cases
- **Cross-component usage**: Used by multiple components
- **Design patterns**: Semantic layer ensures consistent usage

### Examples:

- `color.background.primary` instead of `color.blue.500`
- `spacing.section.large` instead of `spacing.32`
