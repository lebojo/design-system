# T1 Primitive Tokens

## What are T1 Tokens?

T1 tokens are the foundational design tokens that define the raw values of your design system. These are the base values that don't reference other tokens.

### Characteristics:

- **Base values**: Colors, spacing, shadows, etc.
- **No references**: They contain actual values (hex colors, pixel dimensions)
- **Platform agnostic**: Represent the design intent without assumptions about usage
- **Source of truth**: Other tiers reference these values
